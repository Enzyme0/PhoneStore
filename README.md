# Phone Store - OOP Practice

You're going to finish building two classes: `Phone` and `Store`.

Open `Phone.py` and `Store.py` and look for the `TODO` comments. That's where you write your code.

- [ ] `upgrade_storage` - add more GB to the phone
- [ ] `apply_discount` - lower the price by a percentage
- [ ] `add_phone` - put a phone into the store's inventory
- [ ] `sell_phone` - remove a phone from inventory and return it
- [ ] `find_cheapest` - return the cheapest phone in the store

---

To check your work, run:
```
python tests.py
```

Once all tests pass, try running `main.py` to see everything working.

If you're stuck, check out `reference/dog_example.py` - it's a simpler example of the same ideas.
